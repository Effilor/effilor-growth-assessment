# Effilor Logo - IMPORTANT

## Required: effilor-logo.jpg

You need to add your Effilor logo file to this folder as `effilor-logo.jpg`.

### Specifications:

- **Filename**: Must be exactly `effilor-logo.jpg` (case-sensitive)
- **Location**: Root directory (same level as index.html)
- **Format**: JPG/JPEG
- **Recommended size**: 600-800px width
- **File size**: Keep under 500KB for fast loading

### Where to Get It:

Use the same logo file from your previous assessments:
- Strategic Thinking Quotient
- Agile Culture Assessment
- Growth Mindset Assessment

### How to Add:

1. Copy your `effilor-logo.jpg` file
2. Place it in the root of the `growth-assessment` folder
3. Ensure filename is exactly `effilor-logo.jpg`
4. Upload to GitHub with other files

### Verification:

After deployment, the logo should appear:
- Large (300px) on welcome screen
- Medium (120px) on question screens  
- Medium (150px) on results and other screens

### If Logo Doesn't Display:

Check:
- File is named exactly `effilor-logo.jpg` (not .jpeg, not .JPG)
- File is in root directory (not in a subfolder)
- File uploaded successfully to GitHub
